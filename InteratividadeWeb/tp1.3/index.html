Como funciona a Web e o que é uma página web?

A web funciona como uma rede que interconecta computadores e outros dispositivos em escala global para a transferência de dados entre eles. Já as páginas web formam um site e esse site é hospedado em um servidor que é um computador responsável por essa hospedagem.

Quais são os papéis das linguagem HTML e CSS numa página web?

O HTML é a linguagem responsavel por estruturar os elementos da página,como parágrafos, links, títulos, e tabelas, imagens e até vídeos.

Já o CSS é responsavel pelos estilos utilizada para definir cores, fontes, tamanhos, posicionamento e qualquer outro valor estético para os elementos da página.

O que são ambientes de programação?

ambiente de programação também pode ser chamado de ambiente de desenvolvimento. Ou seja, ele é o local onde o trabalho de construção e codificação de um site será realizado.
